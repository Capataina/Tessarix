## Strongest exemplars to study first

If you only had time for three, read these:

1. **Transformer Explainer** (`machine-learning.md`) — multi-pattern composition (multi-level zoom + bidirectional highlight + live model inference). Closest existing analogue to what a polished Tessarix ML lesson should be.
2. **Mathigon** (`mathematics.md`) — locked-prose gating in production. The most extreme version of "you must do to advance" for an entire commercial textbook.
3. **Connected Biology** (`chemistry-biology.md`) — multi-level zoom with causal propagation done well. The pattern for any layered concept (compiler passes, network stack, abstraction in software).
