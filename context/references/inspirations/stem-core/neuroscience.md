### Neuronal Dynamics (EPFL — Gerstner et al.)

- **URL**: https://neuronaldynamics.epfl.ch/
- **What it does**: A free full textbook readable online, with each chapter paired with runnable Python exercises using the Brian Simulator. The exercises simulate Hodgkin-Huxley neurons, vary ion channel conductances, observe action-potential waveforms; simulate spiking networks with STDP and observe synaptic weight evolution; simulate decision-making circuits and vary noise levels. The Brian Simulator runs in-browser via Binder.
- **Interactive pattern**: Live model inference (pattern 14) via Jupyter/Binder + parameter sliders (pattern 7) in the notebook controls.
- **For Tessarix**: The "textbook chapter + immediately runnable simulation of exactly that chapter's model" is the most direct concept-to-code teaching shape. Tessarix's Teach pillar could adopt this structure for any topic with a quantitative model: read the concept, then immediately run the model with adjusted parameters in an embedded sandbox.

### Nicky Case — How To Remember Anything Forever-ish

- **URL**: https://ncase.me/remember/
- **What it does**: An interactive comic that teaches spaced repetition by enacting it. As the reader reads, flashcard prompts appear at spaced intervals — so the reader experiences spaced repetition while learning about it. The decay curve is drawn live as the reader reviews or forgets cards.
- **Interactive pattern**: Concept enacted by the medium (pattern 12) — the strongest exemplar in the catalog of this pattern.
- **For Tessarix**: This is the purest possible version of Tessarix's Quiz pillar — the Teach and Quiz phases merged into a single self-referential artefact. The technique of "demonstrate the concept by making the reader experience it" is a high-priority pattern for any Tessarix lesson on metacognitive or process topics. Worth studying line-by-line as a model for the eventual Tessarix spaced-repetition narrative.
